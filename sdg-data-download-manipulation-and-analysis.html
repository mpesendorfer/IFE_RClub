1. Data Download

The United Nations Department of Economic and Social Affairs provides multiple ways to access the data on Sustainable Development Goal (SDG) indicators.

  • Dashboard: To gain a basic overview over specific indicators, you can use the UNECE Data Dashboard. The Dashboard is a great way to look the definition of an indicator and to compare it across all nations. You can even download the data for specific indicators, but this may be tedious if you intend to use data on multiple indicators.

  • Database Download: To access the full data set, as well as specific subsets of goals or geographic areas, you can access the Global SDGindicators Database. The whole data set consists of more than a million lines (1,174,682 observations as of April 4th, 2020), which will definitely break your Microsoft Excel if try to open the whole file at once. This type of data is usually provided in the .csv format (comma-separated values), which is strongly preferable to a common Excel format. To start, we recommend downloading the data for the country you are investigating. If you do choose to download the whole database, the site will ask for your email address and send you a download link. This is not necessary if select smaller subsets of data.

  • Metadata: The database itself does not provide explanations of the data source, the type of data presented, or the rationale behind a specific SDG indicator. This information is provided as metadata, which can be downloaded from the Metadata repository. Just like in the database, you can download information on all the goals or on a specific subset. The metadata is provided in .pdf and .docx file formats and is downloadable as a zipped folder.

  • Download or transfer the data files or folders to your RStudio Project Folder you created earlier. If all goes as planned, you wont need to open the file directly again.

Investigating Trade-offs and Synergies Between SDG Indicators

Now that we have gained a handle on the data, it is time to pursue the actual goal; looking for relationships among different time series of SDG indicators.

[Note: Time series are a special type of data. For starters, the data points over time are not statistically independent, making direct comparisons a little tricky. On the flipside, there are some great benefits to this type of data. We can observe change over time, investigate how the previous value affects the next one (a phenomenon termed temporal autocorrelation), and much more. This type of analysis is commonly used in pattern analysis and forecasting, e.g.for stock markets, climatology, or ecology. You can make a living analyzing time series!! While this goes beyond the scope of this tutorial and course, I highly recommend learning more about time series analysys, for example here: (https://nwfsc-timeseries.github.io/atsa-labs/)]

Before we are ready to pursue this goal, we need to standardize the data, because a direct comparison of indicator values is often not that meaningful. Standardizing the data means centering and scaling the data so that the mean value becomes zero and the distribution of the data falls within two standard deviations (SD). This allows us to make the comparison A change in 1 SD parameter X was matched by a change in 0.5 SD in parameter Y.

Here, an important point comes up: The direction of the change over time does not always indicate whether we are getting closer to the SDG goal. For example, an increase in the number of women in parliament (see below) is a desired outcome. In contrast, the maternal mortality at birth should decline. For the latter, the statistical trend is negative, but that is actually a good thing. Pay close attention to the desired direction of the indicator change over time! To account for this difference, we will code the data so that the standardized data always increases towards the goal. Therefore, we would invert the standardized values of maternal mortality. We will show you how below.
